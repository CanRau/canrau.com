## Minio on Fly.io

Following the tutorial at [Appkata: MinIO - S3 Compatible Storage](https://fly.io/docs/app-guides/minio/)

### Test locally

```bash
docker run \
  -v storage_volume:/data \
  -p 9000:9000 \
  -p 9001:9001 \
  minio/minio server /data --console-address ":9001"
```

> or use `-v $PWD/data.ignored:/data` to use a relative directory to mount

### Dockerfile

```Dockerfile
FROM minio/minio

CMD [ "server", "/data", "--console-address", ":9001"]
```

### Launch Fly.io app

```bash
fly launch --name "canrau-storage" --dockerfile ./Dockerfile --org personal
```
